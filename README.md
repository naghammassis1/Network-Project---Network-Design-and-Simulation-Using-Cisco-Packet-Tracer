🌐 Network Design & Simulation using Cisco Packet Tracer

📌 Overview

This project designs and simulates a complete network system connecting multiple areas using subnetting, routing, and server configuration.

⚙️ Features

Network design with multiple interconnected areas

IP subnetting for efficient address allocation

Routing using OSPF protocol

Configuration of network services:

DHCP

DNS

Web Server

Mail Server

Wireless and mobile network support

🧪 Functionality

Enables communication between all network segments

Automatically assigns IP addresses using DHCP

Provides web and email services through servers

Tested connectivity using ping and traceroute

🛠️ Technologies Used

Cisco Packet Tracer

Networking (Subnetting, OSPF)

DHCP, DNS

Web and Mail Servers

▶️ How to Run

Open the project file in Cisco Packet Tracer

Start simulation mode

Test connectivity using ping or traceroute

Access web and email services

🎯 Key Concepts

Network design and topology

IP addressing and subnetting

Routing protocols (OSPF)

Network services configuration
Network testing and troubleshooting
